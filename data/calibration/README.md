# data/calibration/

This directory contains the synthetic calibration data and fitted parameters for DTQEM v18.0-C.

## Files

| File | Description | Generated by |
|------|-------------|--------------|
| `synthetic_grid.csv` | Synthetic dataset (I_path, T, C) | `01_generate_synthetic_data.py` |
| `fitted_parameters.json` | Calibrated parameters with uncertainties | `02_loocv_validation.py` |
| `covariance_matrix.npy` | Covariance matrix (NumPy binary) | `02_loocv_validation.py` |

## Data Generation

The synthetic data was generated using:

```python
C = C0 * exp(-a * I_path - b * dT/T_ref - c * I_path * dT/T_ref)
```

with:

· C0 = 0.3675
· a = 1.6968
· b = 0.8055
· c = 0.5000
· T_ref = 300 K

Grid Parameters

· I_path: 21 points linearly spaced from 0 to 1
· T: 21 points linearly spaced from 300 K to 550 K
· Total samples: 441

Reproducibility

To regenerate this data:

```bash
cd scripts
python 01_generate_synthetic_data.py
python 02_loocv_validation.py
```
