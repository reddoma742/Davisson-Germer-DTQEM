# Examples and Sample Outputs

This folder contains representative outputs from the Davisson–Germer + DTQEM simulation code.

## Figures

- `dg_diffraction_T300K.png` – Standard output at room temperature (T=300 K).  
  Shows four panels: diffracted intensity, V(φ) and D(φ), complementarity circle, and θ(φ).

- `dg_diffraction_T500K.png` – Same simulation at T=500 K.  
  Notice the broader Bragg peaks and the reduced visibility V due to increased decoherence γ_t.

- `comparison_T300_T500.png` – Overlay of V(φ) and D(φ) for both temperatures, highlighting the effect.

- `experimental_fit_demo.png` – Illustration of the auto‑calibration procedure: the theoretical V(φ) curve (red dashed) fitted to a simulated diffraction peak (blue solid). The extracted γ_t matches the input value.

## Data

- `bragg_peaks_data_T300.csv` – Numerical values of φ (deg), intensity (norm.), V, D, and V²+D² at T=300 K.  
  Can be used for external analysis or comparison with real experiments.

## How to reproduce

All figures can be regenerated by running the main script with the appropriate parameters. For example:

```python
run_davisson_germmer(energy_eV=54.0, d_nm=0.215, 
                     use_temperature=True, temperature_K=300,
                     auto_calibrate=True, n_max=3)
run_davisson_germmer(energy_eV=54.0, d_nm=0.215, 
                     use_temperature=True, temperature_K=500,
                     auto_calibrate=True, n_max=3)
