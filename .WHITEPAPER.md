# 📄 White Paper – Davisson–Germer + DTQEM

## ⚛️ Abstract

**English**  
This project provides a full simulation of the Davisson–Germer experiment (electron diffraction on a nickel crystal) linked to the DTQEM model. It computes Bragg angles, fringe visibility V, path distinguishability D, complementarity V²+D² ≤ 1, and auto‑calibrates the decoherence rate γ_t from the diffraction peak width. Temperature‑dependent peak broadening and γ_t increase are included. The code is fully vectorised Python, interactive, and supports CSV export.

**العربية**  
يقدم هذا المشروع محاكاة كاملة لتجربة دافيسون-جيرمر (حيود الإلكترونات على بلورة نيكل) مع ربطها بنموذج DTQEM. يحسب الكود زوايا براج، وضوح التداخل V، قابلية التمييز D، التكاملية V²+D² ≤ 1، ويعاير تلقائياً معامل فك الترابط γ_t من عرض قمة الحيود. يشمل تأثير درجة الحرارة (اتساع القمم وزيادة γ_t). الكود متجه بالكامل، تفاعلي، ويصدر بيانات CSV.

## 🔬 Core Physics

**English**
1. **de Broglie wavelength**: λ = h / p, p = √(2mE)
2. **Bragg’s law**: nλ = 2d sin φ
3. **DTQEM mapping**: θ = 180° – 2φ, α = sin(θ/2)=cos φ, V = sinθ·exp(-γ_t), D = |cosθ|
4. **Complementarity**: V² + D² ≤ 1
5. **Temperature effect**: σ(T) = σ₀ + β·(T – T_ref). γ_t increases with T.

**العربية**
1. **طول موجة دي برولي**: λ = h / p
2. **قانون براج**: nλ = 2d sin φ
3. **ربط DTQEM**: θ = 180° – 2φ, α = sin(θ/2)=cos φ, V = sinθ·exp(-γ_t), D = |cosθ|
4. **التكاملية**: V² + D² ≤ 1
5. **تأثير الحرارة**: σ(T) = σ₀ + β·(T – T_ref) ، و γ_t يزداد مع T.

## 💻 Code Structure

- `de_broglie_wavelength(energy_eV)` – compute λ
- `bragg_angles(d_nm, λ, n_max)` – list of (n, φ)
- `gaussian_peak(φ, φ₀, σ, A, bg)` – peak shape
- `dtqem_params(φ, γ_t)` – compute θ, V, D, α
- `calibrate_gamma_t(φ, I, φ_bragg)` – auto‑calibration from peak width
- `run_davisson_germmer(...)` – main simulation, plotting, printing
